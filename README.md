# Planos Peru

## Descripción

Planos Peru Intranet es un proyecto web dedicado a la creación y gestión de planos arquitectónicos. Ofrecemos herramientas y recursos para arquitectos y diseñadores que buscan optimizar su flujo de trabajo.

## Características

- Seguimiento de actividades, tarea y proyectos
- Gestión de expedientes
- Calendario
- Tablero Kanban
- Dashboard
- Gestión de Usuarios

## Tecnologías Utilizadas

- HTML5
- CSS3
- JavaScript / Jquery
- MySQL
- Python / Django

## Requisitos

- Python
- Git

## Instalación

Usando la terminal Git Bash ejecuta los siguientes comandos

1. Clona el repositorio:
   ```bash
   git clone https://github.com/BrianVdeV/integrador
   ```
2. Navega al directorio del proyecto:

   ```bash
   cd integrador
   ```

3. Crea el entorno virtual

   ```bash
   python -m venv .venv
   ```

4. Activa el entorno virtual

   ```bash
   source .venv/bin/activate
   ```

5. Instala las dependencias:

   ```bash
   pip install -r requirements.txt
   ```

## Uso

Ejecuta el siguiente comando en Django

```bash
python manage.py runserver
```

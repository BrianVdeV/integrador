# Planos Peru

## Descripción

Planos Peru Intranet es un proyecto web dedicado a la creación y gestión de planos arquitectónicos. Ofrecemos herramientas y recursos para arquitectos y diseñadores que buscan optimizar su flujo de trabajo.

## Características

- Seguimiento de actividades, tarea y proyectos
- Gestión de expedientes
- Ubicación de Proyectos
- Calendario
- Tablero Kanban
- Dashboard
- Gestión de Usuarios

## Tecnologías Utilizadas

- HTML5
- CSS3
- JavaScript / Jquery
- MySQL
- Python / Django

## Instalación

1. Clona el repositorio:
   ```bash
   git clone https://github.com/BrianVdeV/planosperu
   ```
2. Navega al directorio del proyecto:
   ```bash
   cd intranet
   ```
3. Instala las dependencias:
   ```bash
   pip install -r requirements.txt
   ```

## Uso

Para iniciar el servidor de desarrollo, crea un archivo .env:

```bash
DEBUG=True
DJANGO_ENV=local
```

Y ejecuta el siguiente comando en Django

```bash
python manage.py runserver
```

Abre tu navegador y visita `http://localhost:8000` para ver la aplicación en funcionamiento.
